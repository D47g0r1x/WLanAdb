WLanAdb allows to use standard ADB commands without connecting Android device to desktop by USB cable and without need to root it.

Version 0.1b

How to start using WLanAdb

1 Install desktop client from Downloads section.

2 Check that desktop client and mobile device belong to one subnetwork.

3 Launch WLanAdbTerminal devices from command line and check output.

$ WLanAdbTerminal devices
1 device(s) found:
1) 39334809547D00EC My device - Nexus S (4.1.1)

! If you don't see your device on the list, check that it really belongs to same subnetwork as your desktop.

4 If there is device on list you are ready to use these ADB commands: logcat, push, install.

! Try WLanAdbTerminal help from command line to find out more details.

Security

1 PIN: allows to set keyword to protect from launching WLanAdb and changing its settings.

2 Trusted Hostpots: it is a list of trusted hotspots where WLanAdb service will work. All other hotspots will be ignored. This allows you to automatically enable WLanAdb for home and work networks ignoring public hotspots.
